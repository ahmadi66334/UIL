# UIL Framework

**نسخه ۱.۰.۰**  
**لایسنس MIT**

UIL Framework یک فریمورک CSS مدرن، شیشه‌ای و کشسان است که برای طراحی سریع و زیبای رابط‌های کاربری ساخته شده است. این فریمورک کاملاً بدون جاوااسکریپت کار می‌کند و شامل بیش از ۵۰۰ کلاس کاربردی، ۶ تم رنگی، ۱۰۰ آیکون SVG و سیستم گرید ۱۲ ستونی می‌باشد.

---

## ویژگی‌های اصلی

- افکت شیشه‌ای Glassmorphism با بلور ۲۰px
- انیمیشن‌های کشسان و نرم
- بدون نیاز به جاوااسکریپت
- شش تم مختلف شامل روشن، تیره، بنفش، پرمیوم، مینیمال و شب
- ریسپانسیو و سازگار با تمام دستگاه‌ها
- قابلیت شخصی‌سازی کامل با متغیرهای CSS
- بیش از ۱۰۰ آیکون SVG آماده
- بیش از ۵۰۰ کلاس کاربردی برای هر نوع نیازی

---

## روش‌های نصب

### روش اول: استفاده از لینک CDN

کافی است لینک زیر را در بخش `head` فایل HTML خود قرار دهید:

```html
<link rel="stylesheet" href="https://rozup.ir/view/4276667/UIL-G-V1.0-Final.css">
### روش دوم: دانلود مستقیم فایل

فایل CSS را از لینک زیر دانلود کنید و در پروژه خود قرار دهید: https://rozup.ir/view/4276667/UIL-G-V1.0-Final.css
### روش سوم: نصب از طریق npm

با دستور npm install uil-framework می‌توانید فریمورک را نصب کنید و سپس در فایل اصلی خود آن را import کنید.

مثال سریع:

یک دکمه ساده با کلاس bui bui-primary و یک کارت با کلاس card به همراه عنوان و متن.

شخصی‌سازی:

با تغییر متغیرهای CSS در بخش روت می‌توانید رنگ اصلی، میزان بلور، اندازه گوشه‌ها و سایر تنظیمات را تغییر دهید.
